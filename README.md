Chạy dự án
---
Clone repo về, sau đó chạy dự án bằng cách gõ lệnh 

```
npm run dev. 
```

Để server chạy thành công, lưu ý trên máy phải cài MySQL Server sẳn và đã cho chạy các file sql trong folder server/DatabaseSQL.

Hiện tại server chỉ mới hiện thực API nhân viên và rạp phim, tui sẽ làm tiếp những cái sau.
