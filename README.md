Setup
---

Để server chạy thành công, trên máy phải cài MySQL Server sẳn và đã cho chạy các file sql trong folder server/DatabaseSQL, và tải file .env ở link sau:

```
https://drive.google.com/file/d/16s50Ol68H0ZGo4MISfP5Iznws7NdlquA/view?usp=sharing
```

Sau đó clone repo về và chạy lệnh:

```
npm i
```

Chạy dự án
---
Chạy dự án bằng cách gõ lệnh 

```
npm run dev
```

Server có đủ API (trừ trigger), giao diện employee còn thiếu patch, delete
